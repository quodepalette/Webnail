Advanced: how this works

Capturing a real webpage needs a headless-browser service, since a browser tab can't screenshot another site's page directly. This uses Microlink's free screenshot API — it renders a proper desktop and mobile viewport of the page and hands back real images. Their free anonymous tier is rate-limited; if a capture fails, try again shortly, or upload a screenshot manually instead.
